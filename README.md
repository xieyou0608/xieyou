## 謝侑均個人網站
個人網站連結[https://xieyou.me/](https://xieyou.me/)
使用 Next.js 及 Tailwind CSS 來製作，主要放置網頁工程師及魔術相關的作品
&nbsp;
動畫的部分為了練習，加上不想引入太多第三方套件，全部都是用 Tailwind 手刻，未來可以記錄幾個動畫效果的做法(換頁的升降幕、作品集 hover 字體上升、Timeline接連出現效果、輪播圖等等)
&nbsp;
### Getting Started
```bash
yarn install
# and
yarn dev
```
